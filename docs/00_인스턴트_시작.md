# 00. 인스턴트 시작

이 버전은 Dev Container가 열릴 때 가능한 도구를 자동으로 준비합니다.

## 한 번에 시작

```bash
make start
```

## 1개 작업 자동 실행

```bash
make checkpoint
make auto1
```

## 3개 작업 자동 실행

```bash
make auto3
```

## yolo 모드

Dev Container를 버릴 수 있고 실패하면 다시 풀어서 시작할 생각이면:

```bash
make yolo3
```

## 사람이 알아야 할 최소 개념

```text
next-task   = 다음 작업 선택
codex-run   = Codex가 구현
finish-task = 검증/커밋/병합/완료 표시
auto-loop   = 위 세 단계를 반복
```

## 자동 설치되는 것

- Node 22
- npm / corepack
- git
- bash
- make
- jq
- ripgrep
- GitHub CLI feature
- Codex CLI 설치 시도: `npm install -g @openai/codex`

Codex 인증은 사용자가 한 번 확인해야 할 수 있습니다.
