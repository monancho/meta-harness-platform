# 자동 루프 설명

자동 루프는 다음을 반복합니다.

```text
next-task
→ Codex 실행
→ verify
→ guard-diff
→ commit
→ base branch merge
→ completed marker 생성
```

## 상태 확인

```bash
bash ./scripts/agent/status.sh
```

## 한 작업 진행

```bash
bash ./scripts/agent/auto-loop.sh --limit 1 --mode safe
```

## 여러 작업 진행

```bash
bash ./scripts/agent/auto-loop.sh --limit 3 --mode safe
```

## 실패하면

자동 루프는 실패한 지점에서 멈춥니다.

```bash
bash ./scripts/agent/status.sh
```

필요하면 복구합니다.

```bash
bash ./scripts/agent/restore-checkpoint.sh
```
