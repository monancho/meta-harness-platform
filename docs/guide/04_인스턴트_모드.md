# 인스턴트 모드

## 목표

Dev Container를 열면 필요한 도구를 최대한 자동 설치하고, 사용자는 `./GO.sh`만 실행해 작업을 시작합니다.

## 자동 설치/설정되는 것

- Git, jq, make, ripgrep, zip/unzip 등 기본 도구
- Node 22 환경
- corepack
- npm install
- Codex CLI 설치 시도
- Git safe.directory
- Git user.name / user.email 기본값
- Git pager 비활성화
- Git repo 초기화

## 사용법

```bash
./GO.sh
```

3개 작업:

```bash
./GO.sh 3 safe
```

강한 모드:

```bash
./GO.sh 3 yolo
```

## 왜 로그인은 자동화하지 않는가

Codex 인증은 사용자 계정/토큰과 연결됩니다. 따라서 컨테이너가 Codex CLI를 설치할 수는 있어도, 계정 로그인은 사용자가 직접 승인해야 합니다.

필요 시:

```bash
codex login
```

한 번 로그인한 뒤 다시:

```bash
./GO.sh
```
