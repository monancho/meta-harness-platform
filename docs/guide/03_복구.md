# 복구 가이드

## 최근 checkpoint로 복구

```bash
bash ./scripts/agent/restore-checkpoint.sh
```

## 특정 checkpoint로 복구

```bash
bash ./scripts/agent/restore-checkpoint.sh checkpoint-before-auto-YYYYMMDD-HHMMSS
```

## Git 상태 확인

```bash
git status
git log --oneline --max-count=10
git tag --list 'checkpoint-before-auto-*'
```
