# 처음 시작 가이드

## 목표

이 폴더는 Codex가 Dev Container 안에서 하네스를 단계적으로 구현하도록 만든 작업장입니다.

## 반드시 Dev Container에서 실행

VS Code에서 폴더를 연 뒤:

```text
Dev Containers: Reopen in Container
```

## 첫 명령

```bash
bash ./scripts/agent/autopilot-preflight.sh
```

## 자동 실행

```bash
bash ./scripts/agent/auto-loop.sh --limit 1 --mode safe
```

문제 없으면:

```bash
bash ./scripts/agent/auto-loop.sh --limit 3 --mode safe
```

## 강한 자동 실행

```bash
bash ./scripts/agent/auto-loop.sh --limit 3 --mode yolo
```

단, yolo는 컨테이너와 현재 레포를 버릴 수 있다는 전제에서만 사용합니다.
