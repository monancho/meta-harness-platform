# Portfolio Summary Draft

Meta Harness Platform is a repo-resident project factory for governed AI-assisted delivery.

It demonstrates:

- product architecture for agent workflows
- state-machine driven project generation
- scoped task packets and security boundaries
- deterministic local git worktree execution
- artifact contracts and smoke-tested verification
- dashboard review surfaces without network dependencies
- sanitized feedback loops that avoid storing raw project data

Recommended reviewer commands:

```bash
npm run doctor
npm run verify
npm run demo:e2e
npm run dashboard:preview
```

The project is best evaluated as an MVP DevTool platform skeleton with clear ownership boundaries and extensible execution profiles.
