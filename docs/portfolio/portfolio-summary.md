# Portfolio Summary 초안

Meta Harness Platform은 governed AI-assisted delivery를 위한 repo-resident project factory입니다.

이 프로젝트가 보여주는 것:

- agent workflow를 위한 product architecture
- state-machine 기반 project generation
- scoped task packet과 security boundary
- deterministic local git worktree execution
- artifact contract와 smoke-tested verification
- network dependency 없는 dashboard review surface
- raw project data 저장을 피하는 sanitized feedback loop

추천 reviewer command:

```bash
npm run doctor
npm run verify
npm run demo:e2e
npm run dashboard:preview
```

이 프로젝트는 clear ownership boundary와 extensible execution profile을 가진 MVP DevTool platform skeleton으로 평가하는 것이 적절합니다.
